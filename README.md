## 使用
1. `git clone https://github.com/zhouchong741/jfinal-demo.git`
2. 使用 IntelliJ IDEA 导入项目
3. 安装 Maven 依赖库
4. 启动项目 打开 `localhost` 开始使用吧
5. 数据库文件 `http://zc741.com/jfinal.sql`
